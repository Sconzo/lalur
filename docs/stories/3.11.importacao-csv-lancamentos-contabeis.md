# Story 3.11: Importação de Lançamentos Contábeis via CSV/TXT

**Como** CONTADOR,
**Eu quero** importar lançamentos contábeis via CSV com partidas dobradas,
**Para que** eu possa carregar rapidamente milhares de lançamentos de sistemas ERP.

## Acceptance Criteria

Ver Epic 3 Story 3.11 para critérios completos. Principais pontos:

1. **Endpoint** `POST /api/v1/lancamento-contabil/import`
2. **Formato CSV**: contaDebitoCode, contaCreditoCode, data, valor, historico, numeroDocumento
3. **Validações**: partidas dobradas (débito != crédito), Período Contábil
4. **Performance**: 10k linhas em < 30s

## Definition of Done

- [ ] Endpoint criado
- [ ] Validações partidas dobradas
- [ ] Performance testada
- [ ] Merge em develop
